## Getting started

### Installation

```bash
# Get the latest snapshot
git clone git@github.com:ladaposamuel/pipeline_test.git pipeline_test

# Change directory
cd pipeline_test

# Install NPM dependencies
npm install
or 
yarn install

# Then simply start your app
npm run start:dev
or
yarn run start:dev
```


### Running Build

```bash
# Enter project directory
cd pipeline_test

# Start test
npm run build
or 
yarn run build
```
